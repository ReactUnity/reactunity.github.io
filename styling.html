Font ColorJSX: colorCSS: colorInherited

Sets the text color inside a component.

Valid values

  • Known color name: red, limegreen, transparent
  • Hex color: #a5a5a5, #777, #00ff0033
  • RGBA color: rgba(23, 16, 168, 0.3)
  • CSS keywords: inherit
view {
  flex-grow: 1;
  align-items: center;
  justify-content: center;
  background-color: white;
}


view:nth-child(1) {
  color: crimson;
}


view:nth-child(2) {
  color: #a5a5a5;
}


view:nth-child(3) {
  color: rgba(10, 10, 235, 0.7);
}


view:nth-child(4) {
  color: #550055aa;
}
